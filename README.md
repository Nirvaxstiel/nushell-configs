# maid

Clean up your dev environment.

```
maid -l              list registered tools
maid -c <name>       clean a tool
maid -c -a           clean all
maid -p <name>       prune a tool
maid -p -a           prune all
maid -u <name>       update + clean a tool
maid -u -a           update + clean all
maid -a              clean + prune all
maid -r              probe for installed tools
```

## Install

Clone this repo into your nushell config directory:

```
cd ($nu.config-path | path dirname)
git clone --depth 1 https://github.com/Nirvaxstiel/nushell-configs .
```

Add to your `config.nu`:

```nu
source ($nu.config-path | path dirname | path join ".maid" | path join "init.nu")
```

## Dependencies

Requires `zoxide` and `oh-my-posh`. Your `env.nu` generates their init files:

```
oh-my-posh init nu --config ~/.config/omp/ys.xtended.json | save ...
```

Replace the `--config` path with your own theme, or remove the line if you don't use oh-my-posh.

## Maid only

If you only want maid, strip everything else from `env.nu` and `config.nu`:

**env.nu** — keep only:

```nu
const NU_CONFIG_HOME = ($nu.config-path | path dirname)

mkdir ($NU_CONFIG_HOME | path join ".maid")
```

**config.nu** — keep only:

```nu
$env.config.show_banner = false

source ($nu.config-path | path dirname | path join ".maid" | path join "init.nu")
```

## Adding tools

Edit `.maid/catalog.nu`. Each tool is a record:

```nu
{
  name:     "mytool"
  category: "package-manager"
  detect:   "^where mytool"
  clean:    {|| mytool cache clean }
  prune:    null
  update:   null
}
```

Fields:

- `name` — identifier (e.g. `maid -c mytool`)
- `category` — grouping label (package-manager, toolchain, sdk, etc.)
- `detect` — command checked by probe
- `clean` — closure for `maid -c`, or `null`
- `prune` — closure for `maid -p`, or `null`
- `update` — closure for `maid -u`, or `null`

After editing, run `maid -r` to regenerate the registry.

## Files

- `.maid/catalog.nu` — all known tools
- `.maid/registry.json` — detected tools (generated by `maid -r`)
- `.maid/custom.nu` — user-added tools
